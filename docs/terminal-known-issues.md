## 问题登记模板

新增问题时请包含：

1. 终端类型，以及 Shell 或 CLI 版本。
2. 正式版本是否可复现。
3. 精确操作步骤和会话状态。
4. 涉及渲染时提供截图或短录屏。
5. 预期结果与实际结果。
6. 用户当时位于实时底部，还是正在查看滚动历史。
